## TO-DO APPLICATION
This Project contains a simple TO-DO application.
It has the following features ->
- Anyone can create a TO-DO task.
- Anyone can see their existing TO-DO's
- Anyone can mark their TO-DO's as DONE.

## STACKS USED
- MongoDB
- Express.js
- React.js
- Zod
- Cors
